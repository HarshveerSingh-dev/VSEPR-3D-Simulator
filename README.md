# VSEPR 3D Simulator

Interactive 3D chemistry education platform for visualizing molecular geometries using VSEPR theory.

## Live Demo
https://vsepr-3d-simulator.web.app

## Features

- Interactive 3D molecular visualization
- Real-time rotation and zoom controls
- VSEPR classification system
- Electron geometry visualization
- Bond angle calculations
- Examples for common molecules

## Educational Purpose

Designed to help high school and undergraduate students better understand molecular geometry and spatial arrangements of atoms.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
- Firebase Hosting
